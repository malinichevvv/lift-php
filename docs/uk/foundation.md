# Основа застосунку

> Ця сторінка — **навігаційний хаб**. Її вихідний вміст розбито на спеціалізовані сторінки — кожна тема тепер живе в найпідхожішому для неї місці.

---

## Контролери та маршрутизація

- **[Маршрутизація](routing)** — реєстрація маршрутів, групи, іменовані маршрути, middleware, нормалізація повернених значень.
- **[Маршрутизація через атрибути](attribute-routing)** — `#[Get]`, `#[Post]`, `#[Group]`, `#[Middleware]` на методах контролерів.
- **[Типи обробників](routing#handler-types)** — замикання, `[Class, 'method']`, invokable-класи.

## HTTP

- **[Request](request)** — читання вводу, параметрів, заголовків, cookie, завантаження файлів.
- **[Response](response)** — JSON, HTML, редиректи, стримінг, cookie.
- **[Form Requests](form-requests)** — типізовані, валідовані об’єкти запиту.
- **[JSON-ресурси](json-resources)** — формування виводу API.
- **[Server-Sent Events](sse)** — потокове надсилання із сервера.
- **[HTTP-клієнт](http-client)** — виконання вихідних HTTP-запитів.

## Шаблони та сесії

- **[Шаблони](views)** — PHP-шаблони, макети, секції, часткові шаблони, URL ресурсів, кешування шаблонів.
- **[Сесії](sessions)** — драйвери для файлів, Redis, бази даних і масиву; flash-повідомлення.

## Обробка помилок

- **[Помилки](errors)** — ієрархія винятків, `onError`, `onException`, HTTP-винятки.
- **[Панель налагодження](debug)** — панель, сторінки винятків, зв’язка SQL/логів для локальної розробки.

## Дані

- **[База даних](database)** — Connection, QueryBuilder, Schema, Migrator, Model, SoftDeletes, песимістичні блокування, advisory-блокування.
- **[Валідація](validation)** — усі правила, власні правила, `FormRequest`, `ValidationException`.
- **[Колекції](collections)** — помічник `Collection` із повним довідником методів.
- **[Кеш](cache)** — `ArrayCache`, `RedisCache` з підписом HMAC, адаптер PSR-16.
- **[Файлова система](filesystem)** — `LocalFilesystem`, фасад `Storage`, патерни завантаження.
- **[Redis](redis)** — `RedisClient`, клієнт сирого протоколу, фейки для тестів.

## Безпека

- **[Middleware безпеки](security)** — CORS, CSRF, обмеження частоти, заголовки безпеки.
- **[JWT](jwt)** — підпис/перевірка, middleware, патерн refresh-токенів.
- **[Криптографія](crypto)** — шифрування AES-256-GCM, хешування Argon2id, підпис HMAC.

## Сервіси

- **[Черги](queues)** — задачі, драйвери, воркери, повторні спроби, відстеження провалених задач.
- **[Події](events)** — диспетчер PSR-14, слухачі, події життєвого циклу моделей.
- **[Логування](logging)** — обробники, форматувальники, інтеграція з PSR-3.
- **[Консоль](console)** — CLI-команди, генератори, команда воркера.
- **[Локалізація](localization)** — переклади, множина, перемикання локалі.
- **[JSON-RPC 2.0](json-rpc)** — пакетні запити, маршрутизація методів, коди помилок.
- **[OpenAPI](openapi)** — генерація специфікацій з атрибутів і док-блоків.
- **[Асинхронність (Fibers)](async)** — кооперативна конкурентність із fibers PHP 8.1.

## Довідник

- **[Конфігурація](configuration)** — змінні оточення, `.env`, масиви конфігурації.
- **[UUID та ULID](uuid)** — v4, v7, ULID, бінарне кодування.
- **[DI-контейнер](container)** — прив’язки, синглтони, автозв’язування, контекстна прив’язка.
- **[Тестування](testing)** — `TestCase`, `TestResponse`, помічники запитів.
- **[Порівняння фреймворків](comparison)** — Lift vs Slim vs Lumen vs Laravel Micro.

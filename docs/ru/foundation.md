# Основа приложения

> Эта страница — **навигационный хаб**. Её исходное содержимое разбито на специализированные страницы — каждая тема теперь живёт в наиболее подходящем для неё месте.

---

## Контроллеры и маршрутизация

- **[Маршрутизация](routing)** — регистрация маршрутов, группы, именованные маршруты, middleware, нормализация возвращаемых значений.
- **[Маршрутизация через атрибуты](attribute-routing)** — `#[Get]`, `#[Post]`, `#[Group]`, `#[Middleware]` на методах контроллеров.
- **[Типы обработчиков](routing#handler-types)** — замыкания, `[Class, 'method']`, invokable-классы.

## HTTP

- **[Request](request)** — чтение ввода, параметров, заголовков, cookie, загрузка файлов.
- **[Response](response)** — JSON, HTML, редиректы, стриминг, cookie.
- **[Form Requests](form-requests)** — типизированные, валидированные объекты запроса.
- **[JSON-ресурсы](json-resources)** — формирование вывода API.
- **[Server-Sent Events](sse)** — потоковая отправка с сервера.
- **[HTTP-клиент](http-client)** — выполнение исходящих HTTP-запросов.

## Шаблоны и сессии

- **[Шаблоны](views)** — PHP-шаблоны, макеты, секции, частичные шаблоны, URL ресурсов, кэширование шаблонов.
- **[Сессии](sessions)** — драйверы для файлов, Redis, базы данных и массива; flash-сообщения.

## Обработка ошибок

- **[Ошибки](errors)** — иерархия исключений, `onError`, `onException`, HTTP-исключения.
- **[Отладочная панель](debug)** — панель, страницы исключений, связка SQL/логов для локальной разработки.

## Данные

- **[База данных](database)** — Connection, QueryBuilder, Schema, Migrator, Model, SoftDeletes, пессимистичные блокировки, advisory-блокировки.
- **[Валидация](validation)** — все правила, собственные правила, `FormRequest`, `ValidationException`.
- **[Коллекции](collections)** — помощник `Collection` с полным справочником методов.
- **[Кэш](cache)** — `ArrayCache`, `RedisCache` с подписью HMAC, адаптер PSR-16.
- **[Файловая система](filesystem)** — `LocalFilesystem`, фасад `Storage`, паттерны загрузки.
- **[Redis](redis)** — `RedisClient`, клиент сырого протокола, фейки для тестов.

## Безопасность

- **[Middleware безопасности](security)** — CORS, CSRF, ограничение частоты, заголовки безопасности.
- **[JWT](jwt)** — подпись/проверка, middleware, паттерн refresh-токенов.
- **[Криптография](crypto)** — шифрование AES-256-GCM, хеширование Argon2id, подпись HMAC.

## Сервисы

- **[Очереди](queues)** — задачи, драйверы, воркеры, повторные попытки, отслеживание проваленных задач.
- **[События](events)** — диспетчер PSR-14, слушатели, события жизненного цикла моделей.
- **[Логирование](logging)** — обработчики, форматтеры, интеграция с PSR-3.
- **[Консоль](console)** — CLI-команды, генераторы, команда воркера.
- **[Локализация](localization)** — переводы, множественное число, переключение локали.
- **[JSON-RPC 2.0](json-rpc)** — пакетные запросы, маршрутизация методов, коды ошибок.
- **[OpenAPI](openapi)** — генерация спецификаций из атрибутов и док-блоков.
- **[Асинхронность (Fibers)](async)** — кооперативная конкурентность с fibers PHP 8.1.

## Справочник

- **[Конфигурация](configuration)** — переменные окружения, `.env`, массивы конфигурации.
- **[UUID и ULID](uuid)** — v4, v7, ULID, бинарное кодирование.
- **[DI-контейнер](container)** — привязки, синглтоны, автосвязывание, контекстная привязка.
- **[Тестирование](testing)** — `TestCase`, `TestResponse`, помощники запросов.
- **[Сравнение фреймворков](comparison)** — Lift vs Slim vs Lumen vs Laravel Micro.
